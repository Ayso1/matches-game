# matches-game

Web app where users can play against AI.



## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
## Functional
3 game modes where the user plays against AI. 
1 - the user goes first. 
2 - ai goes first. 
3 - the user goes first and can change the rules of the game.
